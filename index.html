Space Out

I was a key programmer for the Space Out team. I was involved in everything from prototyping to polishing, but here are some standout achievements.

I created and implemented a formula to vary the jetpack's upward force based on the player's current Y velocity. This made it easier to catch yourself when falling and made ascension more controlled, noticeably improving the game feel.
The first implementation of the typing prompts was over-specific to one scene. I refactored it so that most of the same code can be reused in other scenes where gameplay is different, with each scene specifying how it presents the prompts.
